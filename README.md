# MERLIN
### Medical Emergency Routing and Live Intelligence Network

An AI-driven hybrid emergency dispatch system for rural Canada.
Intelligently determines helicopter vs ground ambulance dispatch
using machine learning, mathematical optimization, and safety rules.

---

## 🚀 Features
- AI-based emergency severity prediction
- Travel time estimation using ML models
- Optimization engine for dispatch decisions (PuLP)
- Simulation environment for testing scenarios
- Modular architecture for scalability

---

## 🧠 Tech Stack
- Python 3.11
- scikit-learn
- NumPy, Pandas
- PuLP (optimization)
- Matplotlib / Seaborn

---

## 📌 Status
Active development — Phase 1 (Simulation Engine)

---

## 🎯 Goal
To build a real-world deployable emergency response system
for rural healthcare optimization in Canada.
